About Salesforce AddMe

Salesforce AddMe lets you capture selected text from any web page and push it directly to your Salesforce org as a structured record — Leads, Contacts, Accounts, or any custom object.

Privacy & Data Handling

  • Access tokens are stored in session memory only and cleared when the browser closes.
  • Refresh tokens are encrypted with AES-256-GCM before being written to local storage.
  • Selected text is cached only in session memory for the duration of one workflow.
  • No data is ever sent to third-party servers.

Setting Up a Connected App

  1. In Salesforce Setup, search for App Manager.
  2. Click New Connected App.
  3. Enable OAuth settings and add the Callback URL shown in the form above.
  4. Select the scopes: api, refresh_token, offline_access.
  5. Save and copy the Consumer Key into the Client ID field.
